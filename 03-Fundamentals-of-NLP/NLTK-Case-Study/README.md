# A Case Study of NLTK — Natural Language Toolkit for English
**Subject:** Fundamentals of Natural Language Processing

## Summary
A focused case study exploring the NLTK library for English text processing — covering setup of core NLP packages (NLTK, scikit-learn, langdetect) and their use for tokenization, POS tagging, language detection, and visualization (via matplotlib and wordcloud) on a student habits dataset.
