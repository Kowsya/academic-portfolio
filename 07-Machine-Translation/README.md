# Performance Evaluation of Machine Translation for the Hindi-Konkani Language Pair
**Subject:** Machine Translation

## Summary
Evaluates the current state of Neural Machine Translation for Hindi-to-Konkani, a low-resource language pair. Benchmarks 75 sentences (literal, idiomatic, and sarcastic) across four platforms — Google Translate, Microsoft Translator, AI4Bharat IndicTrans2, and Anuvaad-Bhashini — using the BLEU metric in a Google Colab environment. Anuvaad led among the localized models, though the results highlighted ongoing challenges in capturing Konkani's morphological complexity and cultural nuance.
