# Reinforcement Learning for Protein Folding — HP-Lattice Model
**Subject:** Reinforcement Learning

## Summary
A beginner-friendly technical report applying deep reinforcement learning to protein structure prediction using the HP-Lattice model on the UniProtKB/Swiss-Prot dataset, based on the "Applying Deep Reinforcement Learning to the HP Model for Protein Structure Prediction" paper (ML4PS, NeurIPS 2022). Combines classical and deep RL approaches across roughly 200,000 HP sequences in two phases, bridging molecular biology and AI.

## Contents
- `report.pdf` — full project report
- `colab_notebook.pdf` — exported Colab notebook (PDF)
- `demo/index.html` — interactive 3D visualization of folded structures ([view live](#) once GitHub Pages is enabled — see main repo README)

## Tools & Techniques
Python, Reinforcement Learning, HP-Lattice protein folding model, 3D visualization
